Для развертывания проекта требуется:

1. Открыть проект в Visual Studio 2017
2. Перейти в консоль диспетчера пакета (Package Manager Console) и выполнить следующие команды:
	
	Add-Migration [название миграции] -Project DBRepository - Для создания миграции, чтобы создалась структура БД
	Например: Add-Migration Create -Project DBRepository

	Update-Database - Для применения созданной миграции в БД

3. Запустить программу. 
Тестовые данные типов местоположений, типов оборудования и самих местоположений уже имеются в программе.
Их Insert в БД выполнится при отсутствии данных в таблицах (т.е. при первом запуске)
